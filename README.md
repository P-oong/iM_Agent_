# 🏦 iM Agent
### 🤖 AI 기반 은행 창구 · 영업 시연 웹 데모

고객 조회부터 카드 심사, CRM, KPI, AI 고객 분석까지  
**은행원의 실제 업무 흐름을 하나의 화면 경험으로 연결한 데모 앱**입니다.

> 본 프로젝트의 고객·거래·이벤트 데이터는 **데모용 가상 데이터**이며, 실제 금융기관 시스템과 연동되지 않습니다.

---

## ✨ 핵심 기능

### 🖥️ 통합 뱅킹 시연 화면
- **0156 고객 조회**
- **0310 카드 심사**
- 실제 은행 전산 화면 구조를 반영한 UI
- 고객 조회부터 심사, CRM까지 자연스럽게 이어지는 흐름

### 📊 CRM 사이드 패널
- 고객 보유 상품 및 상태 요약
- **고객 유형별 추천** 제공  
  - 개인
  - 개인사업자
  - 법인
- 영업기회, 금리 우대, AI 인사이트 확인
- 상담형 **AI 채팅 기능** 지원

### 📈 KPI 실적 트랙
- 실적 흐름을 한눈에 볼 수 있는 KPI 바
- 데모 상황에 맞춰 트랙 전환 가능
  - 기본 KPI
  - 💳 마스터카드 이벤트
  - 💰 청약 · 예부금 전환
- 점수 변화와 토스트 알림으로 시연 몰입감 강화

### 🧠 AI 고객 분석
- 가상 고객 선택 후 **AI 자동 분석**
- JSON 기반으로 고객 성향, 추천 상품, 리스크 도출
- **Upstage Solar** 기반 분석 흐름 구현

### 🔗 고객 데이터 연동
- `CustomerContext` 기반 상태 관리
- 고객 조회, CRM, AI 분석 페이지가 동일 고객 기준으로 연결
- **실명번호 앞 6자리 기준 MOCK 고객 연동**
